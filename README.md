\## Overview

Production-style AWS infrastructure implementation covering networking, compute, database, security, and monitoring.



\## Key Components

\- VPC (multi-region)

\- Public \& Private subnets

\- NAT Gateway \& IGW

\- EC2 + Bastion Host

\- Application Load Balancer

\- RDS MySQL + Read Replica

\- Secrets Manager

\- Lambda + SNS alerts

\- CloudTrail + S3 logging

\- VPC Peering

\- DynamoDB (CLI)

\- WAF + CloudWatch
